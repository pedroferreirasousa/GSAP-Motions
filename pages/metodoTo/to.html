GSAP: Método To

É um método que "seta" as configurações finais do elemento.

gsap.to(item, 1.5 , {
x: (index === 0) ? 100 : (index === 1) ? 0 : -100 , 
rotate: (index === 0 ) ? "90deg" : (index === 1) ? "0deg" : 
"-90deg", 
repeat: 1,   
yoyo: true, 
ease: "power2.inOut", 
borderRadius: "50%",
boxShadow: (index === 0) ? "15px 10px 30px 10px rgba(235, 0, 0,
0.5)" : 
(index === 1) ? "10px 10px 30px 10px rgba(46, 230, 0, 0.5)" :  
"10px 10px
30px10px rgba(0, 4, 243, 0.5)",
mixBlendMode: "lighten",
backgroundColor: "transparent", 
});
